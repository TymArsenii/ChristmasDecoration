# CristmasDecoration 
**Effects:**
- Bitmaps(images)
- Snow
- Circles
- Falling commets
- Sine wave

**Single button control:**
- Single - next effect
- Double - toggle auto effect change
- Tripple - enter favorite effects menu:
  - Single - next effect id
  - Hold - toggle effect
  - Tripple (again) - exit
- Hold - brightness
